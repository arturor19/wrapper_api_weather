## Preconditions:

- Python 3

## Clone the project

```
git clone git@github.com:arturor19/wrapper_api_weather.git
```

### Move to the wrapper_api_weather folder

```
cd wrapper_api_weather
```

### Install dependencies

```
pip install -r requirements.txt
```

### Run test

```
pytest
```


### Run server

```
uvicorn main:app --reload
```
